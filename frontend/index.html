Installation

ConsentOS is a Chrome extension that runs locally in your browser. Follow these steps to get started:

Step 1: Get a Gemini API Key

  1. Visit Google AI Studio
  2. Create a free API key (no credit card required)
  3. Copy your API key

Step 2: Configure the Extension

  1. Download or clone the ConsentOS repository
  2. Open my-extension/background.js
  3. Replace YOUR_GEMINI_API_KEY_HERE with your actual API key
  4. Save the file

Step 3: Load the Extension

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked"
  4. Select the my-extension folder
  • ConsentOS is now active!

    • That's it! Now when you visit websites with consent popups, ConsentOS will automatically detect and analyze them, showing you a clear breakdown of what you're agreeing to.

    1. Detection: ConsentOS monitors web pages for consent popups and cookie banners using pattern matching and DOM observation.

    2. Extraction: When a popup is detected, the extension extracts the consent text and related privacy policy URLs.

    3. Analysis: The text is sent to Google Gemini AI, which analyzes it and provides a structured breakdown including:

    • Plain-English summary of what you're agreeing to
    • Specific concerns and potential risks
    • Types of data that will be collected
    • Clear recommendation (Accept/Decline/Review)

    4. Display: ConsentOS shows you this analysis in a beautiful overlay before you make a decision, empowering you to give truly informed consent.
